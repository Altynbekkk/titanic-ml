# Titanic Survival Prediction

This project predicts passenger survival on the Titanic using machine learning.

## Model
Random Forest / Gradient Boosting

## Features
- Pclass
- Age
- Fare
- Sex
- FamilySize
- Embarked
- Title

## Result
Kaggle score: 0.76

## Libraries
- pandas
- numpy
- scikit-learn
- xgboost
